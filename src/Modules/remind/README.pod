=head1 NAME

limebar-remind - remind module for limebar.

=head1 DESCRIPTION

B<limebar-remind> is a module for limebar. It set label B<remind_nb> (number of reminders active).

=head1 RESOURCES

limebar-remind add some config option to the config files:

=over

limebar.remind.file:        		path to the files to use for remind.

=back
